# Nat_CV

## 📋 Présentation du projet

**GRWP-CV-Nat** est une application web destinée à la création et à la gestion de CV en ligne. Ce projet est entièrement développé en **HTML** afin de garantir une simplicité d'accès et de personnalisation pour tout utilisateur. L'objectif principal est de fournir une interface intuitive permettant la visualisation, la modification et la mise en ligne d'un CV professionnel.

### ✨ Fonctionnalités principales

- ✅ Affichage dynamique des informations du CV
- ✅ Code entièrement en HTML pour une compatibilité maximale
- ✅ Structure modulable pour une personnalisation facile
- ✅ Déploiement rapide du site en ligne
- ✅ Interface responsive et moderne

---

## 🔄 Workflow Git employé

Le projet GRWP-CV-Nat utilise **GitKraken** comme outil de gestion de versions et de collaboration.

### Pourquoi GitKraken ?

GitKraken facilite :
- La visualisation graphique de l'historique Git
- La gestion intuitive des branches et des merges
- La collaboration en équipe de manière productive
- La détection facile des conflits de fusion

### 📝 Étapes du workflow

1. **Création de branches** : Une branche pour chaque nouvelle fonctionnalité ou correction de bug
2. **Commits réguliers** : Messages clairs et descriptifs à chaque modification
3. **Revue visuelle** : Utilisation de GitKraken pour examiner les changements avant fusion
4. **Tests locaux** : Vérification des modifications en local
5. **Push et publication** : Envoi vers le dépôt GitHub et déploiement en ligne
